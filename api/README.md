## Setup

1. **Configure environment variables:**
   - Copy `.env.example` to `.env`
   - Update your database credentials in `.env` (DB_USER, DB_PASSWORD, DB_NAME)

2. **Run the backend:**
   In the api folder:
   ```bash
   # on first run
   npm install

   npm run dev
   ```

3. **Run the frontend:**
   In the client folder:
   ```bash
   # on first run
   npm install

   npm start
   ```
